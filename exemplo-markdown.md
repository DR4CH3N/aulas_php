## Exemplos de codigo no markdown
com 2 ` é um codigo de apenas uma linha

`console.log("Olá!")`

com 3 ` são multiplas linhas
```javascript
$aluno = "thiago";
echo $aluno;
```

```php
$aluno = "roberto";
echo $aluno;
```
